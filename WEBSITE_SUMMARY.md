# Rina Resuello - Premium Consulting Website

## Project Overview

A professional, premium personal consulting website for Rina Resuello, an Amazon Catalog Troubleshooter and Account Health Expert. The website is designed to position her as a high-level consultant for Amazon sellers, private label brands, aggregators, e-commerce agencies, and brand owners selling on Amazon US.

## Animations & Interactions

- **Smooth Scrolling:** All navigation links and buttons use smooth scroll behavior
- **Fade-in Animations:** Elements animate in as they come into view
- **Button Hover Effects:** Buttons lift and show shadow on hover
- **Counter Animations:** Statistics animate when they come into view
- **Form Validation:** Contact form validates input and provides user feedback
- **Responsive Design:** Mobile-first approach with breakpoints for tablets and desktops

## Technical Stack

- **HTML5:** Semantic markup
- **Tailwind CSS 4:** Utility-first CSS framework
- **JavaScript:** Vanilla JS for animations and interactions
- **Google Fonts:** Poppins and Inter fonts
- **Responsive Design:** Mobile, tablet, and desktop optimized
